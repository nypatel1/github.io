The Agentic Build — Python Quiz App

HW8

An exercise in spec-driven development with AI agents. I wrote a detailed specification for a Python quiz application, then handed it to GitHub Copilot to build autonomously. The resulting app features user authentication, weighted question selection based on feedback, a hint system with half-point scoring, and SQLite-backed stats with ASCII bar charts. After the build, an AI-powered code review evaluated the output against acceptance criteria, followed by a personal reflection on where agentic workflows shine and where they fall short.

Key Aspects:

  • Wrote a full product spec with acceptance criteria before any code was written
  • AI agent (GitHub Copilot) built the entire codebase from the spec autonomously
  • Quiz engine with weighted random selection, category filtering, and hint scoring
  • Secure user authentication with hashed passwords and login attempt limits
  • Score tracking with per-user stats and ASCII bar chart history
  • AI code review identified gaps between spec and implementation
